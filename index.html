VIBE CODING TRACKS

바이브코딩은 AI에게 원하는 걸 말하고, AI가 만든 결과를 직접 실행하고 확인하면서 제품을 완성해가는 방식입니다. 이 페이지는 개발 동아리 신입부원을 기준으로, 당신에게 맞는 진입 트랙을 고를 수 있게 설계된 업그레이드 버전 안내서입니다.

SCROLL TO BEGIN
What Is Vibe Coding

바이브코딩은 “설명 → 생성 → 실행 → 피드백”의 반복입니다

기존 개발은 언어 문법을 배우고, 직접 코드를 설계하고, 필요한 모든 구조를 처음부터 쌓는 흐름이 강했습니다. 반면 바이브코딩은 아이디어를 먼저 말하고 AI가 초안을 만들면, 사람이 그 결과를 검토하고 다듬는 루프를 빠르게 반복합니다.

01

설명한다

무엇을 만들고 싶은지, 어떤 화면과 기능이 필요한지 자연어로 먼저 정리합니다.

02

생성한다

AI가 초안 코드를 만들고, 여러 파일과 구조를 빠르게 제안합니다.

03

검증한다

실행, 화면 확인, 에러 해결, 배포까지 사람이 끝까지 책임지고 판단합니다.
