바이브코딩이란?

먼저 개념부터 이해해봐요

기존 개발 방식은 프로그래밍 언어 문법을 익히고, 직접 코드를 작성하는 방식이었습니다. 바이브코딩은 이 과정을 뒤집습니다. 내가 만들고 싶은 것을 말로 설명하고, AI가 코드를 만들어주면 내가 검토하고 실행해보는 방식입니다.

처음에는 "이게 진짜 개발인가?" 싶을 수 있어요. 하지만 실제로 동작하는 서비스를 만들면서 개발의 흐름을 몸으로 익히게 됩니다. 완벽한 코드보다 움직이는 결과물을 먼저 만드는 게 목표입니다.

01

요청

만들고 싶은 기능을 최대한 구체적으로 AI에게 설명합니다. "로그인 버튼 만들어줘"보다 "파란색 로그인 버튼, 클릭하면 콘솔에 메시지 출력"처럼요.

02

확인

AI가 만든 코드를 브라우저에서 직접 실행해봅니다. 잘 되면 다음 단계, 이상하면 오류 메시지를 AI에게 그대로 전달합니다.

03

반복

요청 → 확인 → 수정을 반복하며 기능을 하나씩 완성해 나갑니다. 마지막엔 GitHub에 올리고 Vercel로 배포해서 실제 링크를 만들어요.
